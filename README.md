# ASG Monitor

Minimal monitoring repository for Infinity Empire services.

## Purpose

This repository is prepared to hold uptime checks, deployment health checks, and automation monitoring scripts.

## Current status

The repository currently contains monitoring documentation only. Runtime monitor scripts still need to be added:

- `monitor.js` for Node.js service health checks.
- `package.json` with `npm run check`.
- Optional cron workflow for scheduled checks.

## Target services

- Delta Agent
- Social API Hub
- Lottery Bot
- Tal HaTil website
